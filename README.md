# SpringDAO-TheoriqAI-Auto-Bot
**此项目仅供学习，不作为任何商业盈利手段，如有非法使用均与本人无关！**

# 功能说明

1. 自动根据钱包私钥注册
2. 自动绑定twitter和discord

# 使用说明

1.在config.yaml文件中进行配置

```yaml
#theoriq AI配置
theoriq:
  discord_token: '你的discord token'
  twitter_token: '你的推特token'
  invite_code: '你的邀请码'
  private_key: '你的钱包私钥'
```

2.双击theoriq_bot.exe启动，根据菜单说明使用
